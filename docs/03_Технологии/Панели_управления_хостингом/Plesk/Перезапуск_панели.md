# Запуск и перезапуск Plesk
## Запустить Plesk
``` bash
sudo /etc/init.d/psa start
```

## Перезапуск Plesk

``` bash
/etc/init.d/psa restart
```
или

``` bash
/etc/init.d/psa stop
/etc/init.d/psa start
```

## Остановка Plesk
``` bash
/etc/init.d/psa stop
```
