# Монтируем раздел, смонтированный в ro в режиме rw(чтения-записи)
Используем команду
``` bash
sudo mount -o remount,rw /dev/sdaX /
```
где `/dev/sdaX` - нужный разел, а `/` - каталог куда смонтирован данный разел.
